# meshcoresim.com-releases

DE

Dieses Repository enthält die veröffentlichten Artefakte:

- Android APKs (Dev/Prod)
- Desktop Builds für Windows, macOS und Linux

Hinweis: GitHub erstellt fuer jedes Release automatisch "Source code (zip/tar.gz)".
Diese enthalten nur den Inhalt dieses Repos (z.B. README/LICENSE) und nicht den
privaten App-Quellcode.

EN

This repository contains the published artifacts:

- Android APKs (Dev/Prod)
- Desktop builds for Windows, macOS, and Linux

Note: GitHub automatically adds "Source code (zip/tar.gz)" to every release.
These archives only include the contents of this repo (e.g. README/LICENSE),
not the private app source code.
